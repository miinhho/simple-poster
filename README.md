Poster website without backend.    

Default backend url is http://localhost:8080.    
Based on the code in [Next.js 15 & React - 완벽 가이드](https://www.udemy.com/course/nextjs-react-incl-two-paths).    

But I used TypeScript, add I changed some styles with tailwindcss.    
Also some codes are changed to fit my code style.    
Lastly, many codes are changed. DON'T THINK THIS IS THE SAME AS THE COURSE.    


## Important
- This project is not for production. It's just for studying.
- You need to make '.env' file in the root directory and write 'BACKEND_URL' in it. (e.g. BACKEND_URL=http://localhost:8080)
- You need to run backend server before running this project.
